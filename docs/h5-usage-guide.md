# H5页面使用帮助指南

## 1. 指南概述

本指南将帮助您快速了解和使用律师事务所管理系统的H5页面，包括案件查询、进度跟踪、消息查看等核心功能。

## 2. 登录与注册

### 2.1 登录系统

1. 打开H5页面，您将看到登录界面
2. 在「用户名」输入框中输入您的账号
3. 在「密码」输入框中输入您的密码
4. 如需下次自动登录，可勾选「记住密码」
5. 点击「登录」按钮进入系统

### 2.2 忘记密码

1. 在登录页面点击「忘记密码？」链接
2. 按照提示输入相关信息，重置密码
3. 使用新密码重新登录

## 3. 首页（我的案件）

登录成功后，您将直接进入首页。首页展示了您的案件概览和重要信息：

### 3.1 案件统计
- **我的案件**：显示您的所有案件总数
- **进行中**：显示当前正在处理的案件数量
- **已完成**：显示已结案的案件数量
- **待处理文书**：显示需要您处理的文书材料数量

### 3.2 最近案件动态
- 显示您的案件最新进展和状态变化
- 点击任意动态可直接跳转到对应案件详情

### 3.3 进行中案件
- 列表形式展示当前正在处理的案件
- 显示案件名称和当前状态
- 点击案件可进入详情页

### 3.4 待办事项
- 显示您需要完成的任务列表
- 包含任务标题和截止日期
- 可点击复选框标记任务为「已完成」
- 点击任务可查看详情

### 3.5 快捷操作
- **在线咨询**：快速与律师进行沟通
- **提交材料**：上传案件相关材料
- **查看文书**：快速访问案件文书

## 4. 案件管理

### 4.1 案件列表

1. 点击底部导航栏「案件查询」图标进入案件列表
2. 使用顶部搜索框可按关键词搜索案件
3. 点击标签页可筛选不同状态的案件：
   - **全部**：显示所有案件
   - **待处理**：显示等待处理的案件
   - **进行中**：显示正在处理的案件
   - **已完成**：显示已结案的案件
4. 点击任意案件卡片可进入案件详情

### 4.2 案件详情

进入案件详情页后，您可以查看：

#### 4.2.1 案件基本信息
- 案件编号、标题、类型、状态
- 负责律师、立案时间、预计结案时间

#### 4.2.2 案件进度
- 以时间轴形式展示案件的完整处理流程
- 清晰显示当前案件所处阶段
- 包含每个阶段的描述和时间

#### 4.2.3 文书材料
- 列表展示案件相关的所有文书文件
- 显示文件名和上传时间
- 点击文件可在线预览或下载

#### 4.2.4 待办事项
- 显示该案件相关的待办任务
- 可标记任务完成状态
- 显示任务截止日期

#### 4.2.5 联系律师
- 点击底部「联系律师」按钮
- 可直接与负责该案件的律师进行沟通

## 5. 消息中心

### 5.1 查看消息

1. 点击底部导航栏「消息中心」图标
2. 查看所有系统通知和律师消息
3. 未读消息右上角会显示红色数字提示
4. 点击消息可查看详细内容

### 5.2 消息类型
- **案件进度更新**：案件状态变化通知
- **律师通知**：律师发送的重要信息
- **系统消息**：系统相关通知

## 6. 个人中心

### 6.1 个人信息

1. 点击底部导航栏「个人中心」图标
2. 查看您的头像、姓名和角色
3. 可点击进入详细个人信息编辑

### 6.2 功能服务
- **案件进度**：快速查看案件进展
- **咨询预约**：预约律师咨询
- **消息中心**：进入消息列表
- **资料上传**：上传案件相关资料

### 6.3 设置选项
- **个人信息**：编辑您的基本信息
- **安全设置**：修改密码、绑定手机等
- **通知设置**：管理消息通知偏好
- **关于我们**：了解系统相关信息

### 6.4 退出登录

1. 在个人中心页面底部
2. 点击「退出登录」按钮
3. 确认后将退出当前账号

## 7. 常见问题

### 7.1 如何修改密码？

1. 进入个人中心
2. 点击「安全设置」
3. 选择「修改密码」
4. 按照提示输入原密码和新密码
5. 确认修改

### 7.2 如何联系客服？

1. 进入个人中心
2. 点击「关于我们」
3. 查看客服联系方式

### 7.3 如何查看案件历史记录？

1. 进入案件详情页
2. 在「案件进度」部分查看完整历史记录
